# hoshi-no-yashiro（星の社）

3D創世瞑想アプリ。触ると光の塵が集まって星が生まれ、生まれた神様が宇宙を漂う。
クロちゃんデザインの2D版「八百万の神々」を3D化する構想の Phase 1。

## 遊び方

* `index.html` をブラウザで開くだけ（ビルド不要。Three.jsはCDNから読む＝ネット接続が必要）
* お参り: 呼吸モードを選び、光のリングに合わせて3呼吸 → 画面に8秒触れて星を迎えると神様が誕生
* 奉納の舞: 縁を結んだ神様が円座に並び、太鼓に合わせて光の環が近づく。環が重なる刹那にタッチすると祝詞が届く
* 右上の🔇でヒーリング音のON/OFF（音はコード生成、音声ファイルなし）
* スマホのタッチでも同じ操作

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 本体。HTML1枚に全部入り（CSS/JS込み） |

| `docs/original-design.html` | クロちゃん作の2D版元デザイン（8.7MB・AIは全文を読まないこと） |

## 開発を引き継ぐAIへ

1. まず `docs/CONCEPT.md` を読む（構想の正本）
2. `index.html` の中身はセクションコメントで区切ってある
   （基本セットアップ／星空／塵／神様／種／入力／音／呼吸ガイド／メインループ）
3. 動作確認は `.claude/launch.json` の `hoshi-no-yashiro` でプレビューサーバーを起動
4. 次にやることは CONCEPT.md の「決まっていないこと」参照

* Phase 2 公開済み（2026-07-13・クロちゃん/Fable 5）
