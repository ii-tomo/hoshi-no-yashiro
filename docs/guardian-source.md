# 守護神を導き出す方法と各神様の特徴
 
あなたを守護している神様を計算で見つける方法と、各神様の特徴をまとめました。
 
## 計算方法
 
1. 生年を1桁ずつ足す
2. 計算結果に生まれた月を足す
3. 計算結果に生まれた日を足す
4. 計算結果が2桁なら、さらに足して1桁に
5. ただし、最後の計算で11、22、33になったら、そのままの数字を使う
 
## 計算例
 
| 例：1997年12月22日の場合 | 計算過程 |
|--------------------------|----------|
| 生年を1桁ずつ足す | 1+9+9+7 = 26 |
| 計算結果に月を足す | 26+12 = 38 |
| 計算結果に日を足す | 38+22 = 60 |
| 2桁を1桁に変換 | 6+0 = 6 |
| 最終結果 | 6 |
